# 병목 점수 가중치 재추정 기록 (2026-07-17)

### ② 길이(어절수) 통제 후 각 자질의 순수 기여 — 다중회귀 `총시간 ~ 어절수 + 4자질`

## 2. 구인 타당도 논의 반영

- **절밀도**는 (a) 안긴절과 중복되고, (b) 순수 기여가 없으며, (c) 값을 낮추는 방식이
  '문장 쪼개기'라 응집성(cohesion) 저하라는 다른 종류의 부담을 끌어들인다. → 원칙적으로 제외 대상.
- 그러나 실제로 **절밀도를 완전히 제거하면 thermal-4·osmosis-5 명제에서 하<중<상 순서가 깨진다.**
  이 명제들의 '하' 버전은 문장을 쪼개 만들어 **안긴절 수가 '중'보다 오히려 많기** 때문이다
  (= 쪼개기가 통사 복잡성을 깨끗이 낮추지 못한 자료상의 결함). 그래서 절밀도를 **소량(0.3)만 유지**한다.
